Welcome To RoboStock!

This interactive Web Application seeks to demonstrate to users one of the many Machine Learning applications. By using an LSTM (Long-Short Term Memory) Machine Learning Algorithm, this application attempts to predicts stock prices based on the historical closing price time-series data for the selected stock. The historical closing stock price is queried from Yahoo Finance. Scikit-learn and Keras are the machine learning libraries used for executing the LSTM algorithm. The graphs are rendered using Google Charts in order to provide the users with some level of interaction.
